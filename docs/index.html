Contexto

Detalhes sobre o espaço de problema, justificativas e os objetivos do projeto.

Problema

Os ataques cibernéticos representam uma ameaça significativa para empresas, organizações e indivíduos em todo o mundo. Eles podem resultar em roubo de dados, interrupção de serviços, prejuízos financeiros e comprometimento da privacidade. Algumas formas comuns de ataques incluem roubo de dados, ataques de negação de serviço (DDoS), phishing e engenharia social, exploração de vulnerabilidades de software e ataques a infraestruturas críticas. Para mitigar essas ameaças, é essencial que as empresas implementem medidas robustas de segurança cibernética, como firewalls, antivírus, sistemas de detecção de intrusão e políticas de segurança bem definidas.Esses ataques visam informações pessoais, financeiras e até mesmo dispositivos domésticos conectados à Internet. Para se proteger, é importante adotar práticas de segurança digital, como usar senhas fortes, manter software atualizado e uso de antivírus

Objetivos

O objetivo geral deste trabalho é desenvolver um software para combater e mitigar os efeitos dos ataques cibernéticos direcionados a pessoas comuns. Esse software terá como objetivo principal proteger informações pessoais, financeiras e dispositivos domésticos conectados à Internet contra ameaças online. O primeiro objetivo deste trabalho é desenvolver um software dedicado a verificar a confiabilidade dos links na web, visando proteger os usuários contra ataques de phishing e outras ameaças cibernéticas. Essa abordagem busca fornecer aos usuários uma ferramenta eficaz para avaliar a segurança dos URLs antes de clicarem neles, contribuindo assim para aumentar a segurança online. O segundo objetivo específico é criar um site de conscientização sobre segurança cibernética, com o propósito de educar o público sobre os riscos e melhores práticas para se proteger contra ataques online. Isso envolve a produção de conteúdo educativo, como artigos, vídeos e infográficos explicativos sobre os tipos de ataques cibernéticos e suas consequências, juntamente com informações sobre como os usuários podem se proteger. O objetivo é alcançar uma ampla audiência e oferecer recursos práticos para aumentar a conscientização e capacitar os usuários a se protegerem contra ameaças online.

Justificativa

A decisão de trabalhar no desenvolvimento de um software para verificar a confiabilidade de links e criar um site de conscientização sobre segurança cibernética foi motivada pela necessidade urgente de enfrentar os crescentes desafios dos ataques cibernéticos na era digital. A maioria dos entrevistados na pesquisa não foi diretamente vítima de ataques cibernéticos, mas reconheceu casos de amigos, familiares ou conhecidos afetados por essas ameaças. Isso ressalta a importância crítica de adotar medidas proativas para fortalecer a segurança cibernética e proteger-se contra ameaças online. Os objetivos específicos foram escolhidos com base na análise das necessidades identificadas durante as entrevistas e na compreensão das lacunas de conhecimento e áreas de preocupação relacionadas à segurança cibernética. A decisão de desenvolver um software para verificar a confiabilidade de links visa proteger os usuários contra ataques de phishing e outros tipos de malware, oferecendo uma solução prática para tomar decisões informadas sobre a segurança dos links online.

Público-alvo

Nossa aplicação será utilizada por uma ampla gama de usuários, incluindo jovens conectados, idosos aprendendo tecnologia, pais preocupados e pequenos empresários. Os jovens conectados são proficientes em tecnologia, mas podem não estar cientes dos riscos cibernéticos. Os idosos aprendendo tecnologia estão começando a utilizar dispositivos digitais e buscam recursos intuitivos para se proteger online. Os pais preocupados estão interessados em ferramentas para proteger seus filhos online. Os pequenos empresários valorizam soluções práticas para proteger seus negócios online. É importante considerar as diferentes necessidades e níveis de conhecimento desses grupos ao projetar nossa aplicação.

Concepção (Design Thinking)

Detalhes do processo de discovery do projeto.

Durante o processo de descoberta do projeto, realizamos uma pesquisa abrangente para compreender o cenário da segurança cibernética, entrevistamos usuários para obter insights valiosos sobre suas necessidades e expectativas. Desenvolvemos personas representativas dos diferentes perfis de usuários. Com base nessas informações, definimos objetivos claros para o projeto e iteramos continuamente nossas ideias com base no feedback recebido. Esse processo de descoberta foi essencial para estabelecer uma base sólida para o desenvolvimento da aplicação, garantindo que nossas soluções sejam centradas no usuário e eficazes na abordagem dos desafios da segurança cibernética.

Solução

Esta seção apresenta todos os detalhes da solução criada no projeto.

Lições: Criamos uma aba de Lições para que o usuário consiga ler e aprender sobre algum tipo de ataque cibernético, a estrutura associada foi um setor com vários ícones de cada tema e com exemplos claros de como funcionam estes ataques.

Quiz: Criamos um quiz interativo na qual o usuário pode optar por fazer ou não no final de cada lição, a estrutura associada são cerca de 3 perguntas e no final o usuário ira saber quantas ele acertou. Cada quiz tem um tema diferente dependendo da lição escolhida.

Potencializador de senhas: Criamos uma pagina na qual voce pode testar suas senhas para saber o potencial delas. Dependendo dos requisitos cumpridos, o potencializador ira dizer se a senha sera forte, media ou fraca.

Gerenciador de senha: Criamos uma pagina na qual você pode armazenar suas senhas. Você pode escolher se quer manter aquela senha, adicionar outras ou simplesmente apagar.

Verificador de links: Criamos uma aba na qual você pode testar se um link sera fraldoso. Colando o link na aba você receberá o resultado

Chatbot: Criamos uma pagina na qual possui um bot que pode te auxiliar. Dependendo das opções que você escolher, ele irá te sugerir explicações sobre.

FAQ

Como acessar as funcionalidades do projeto?
Para acessar o projeto certifique-se de ter posse de um editor de arquivos ou IDE para uma melhor experiência. Abra o navegador e acesse o localhost, ou, através do Visual Studio Code, baixe a extensão liver server e acesse o site por ela.
Quais recursos e ferramentas são necessários para o acesso da aplicação?
É necessário o acesso a um browser com conexão à internet. Através do seu buscador procure por localhost ou acesse a extensão liver server do Visual Studio Code.
A aplicação passar por teste de segurança e verificações de falha?
Até o momento, não. O site está fase de desenvovimento e por enquanto estamos nos atentando ao testes de funcionalidades básicos. Até a entrega final esses aspectos serão colocados em prática.
